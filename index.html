Emoji Bitcoin wallet

Tired of boring 12-word seeds? Use emoji instead! The same security 🔑 in a funny form 😻!

⚠️ Warning: using emoji seed is not very safe as there are plenty of similar emojis (for example 💀 and ☠️) and they are platform-dependent. But you still can do it if you want 😈.

Your emoji seed:

Refresh this page to get a new seed.

To detect typos we use a wordlist-independent electrum checksum approach: Emoji seed generator creates sequences of random emojis and checks if their HMAC starts with "01" (electrum standard seed). With this type of checksum even using a different list of emojis will lead to a valid seed, and if you've made a typo when entering the seed first two characters of the HMAC will be different from "01", so you will know that you made a typo.

Seed hash:

Hash should start with "01". It's a simple check to make sure you didn't make a typo.

 
 

Your master private key:

You can import it to any wallet. Derived the same way as in electrum wallet.

 
 

Your master public key:

You can import it to any wallet to watch your funds. Derived the same way as in electrum wallet.
